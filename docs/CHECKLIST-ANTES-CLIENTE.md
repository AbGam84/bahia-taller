# Katire — checklist antes de mostrar al cliente

## Obligatorio (2 minutos)

1. Ejecutar `Verificar-Antes-Cliente.bat`  
   Debe decir: **SMOKE OK**
2. Abrir https://katire.onrender.com y entrar con `admin` / `DuenoKatire2026`
3. Probar en este orden:
   - Ingreso (placa + queja; foto/firma opcionales con confirmación)
   - Diagnóstico & OT (semáforo + guardar OT)
   - Imprimir diagnóstico
   - Repuestos & tiendas (buscar en Gigante/Guacamaya)
   - Aliados (registrar aliado + seguimiento)
   - Facturación: checklist “Listo para Hacienda”

## Facturación real (solo si va a emitir)

- Usuario ATV + clave ATV
- Certificado `.p12` + PIN
- Ambiente: Sandbox primero
- Emitir desde OT → debe firmar, enviar y consultar aceptación

## No entregar si

- El bat de verificación falla
- Patio en vivo da error al cargar
- Login no acepta admin
- Falta el `.p12` y el cliente espera “aceptado por Hacienda” hoy
